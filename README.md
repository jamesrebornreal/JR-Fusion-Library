# Fusion Library

A focused library designed to make Fusion mod development easier, faster, and more enjoyable.

## About

Fusion Library is a collection of useful functions and utilities originally developed for **Fusion Protector**. The library has been cleaned up, refined, and made publicly accessible so developers can reference it, build upon it, and create their own mods for Fusion.

While I no longer actively play Fusion, I wanted to give back to the community by sharing tools that can help other developers create better experiences, experiment with new ideas, and make mod development more approachable.

## Features

* Simplified utility functions for mod development
* Reusable code to reduce development time
* Helpful tools for common Fusion modding tasks
* Designed to make creating mods cleaner and more enjoyable
* Community-focused and open for developers to build upon

## Getting Started

To use Fusion Library:

1. Add the library reference to your project.
2. Import the required namespaces.
3. Use the provided functions and utilities to help speed up development.

Example:

```csharp
using FusionLibrary;

// Your mod code here
```

## Purpose

This project exists to support the Fusion modding community by providing a foundation that developers can use when creating their own projects.

Whether you're making your first mod or working on a larger project, Fusion Library aims to make the development process smoother and more enjoyable.
